<p><p>Back glue markets are categorized into three primary types based on composition and application. </p><p>1. **1 Component Paste BG**: This consists of a single formulation, ready for use, offering convenience in application for bonding materials. </p><p>2. **1 Component Liquid BG**: Similar to the paste, this liquid version is easy to handle and provides effective adhesion with a quick-drying time. </p><p>3. **2 Component BG**: This type requires the mixing of a resin and hardener before application, resulting in a stronger bond, suitable for demanding industrial uses.</p></p>
